Inventory Menu

This is an inventory main menu from which you can access the full inventory. It can be accessed via:

  • http://localhost:8080, the default index.html page for the website.
  • or, http://localhost:8080/index.html if the index.html document name is included in the URL.

The following link brings you to the full inventory list:

  • List of Items. The JSP page is a list of all the categories of items within the inventory.
